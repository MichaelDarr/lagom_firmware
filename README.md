# lagom_firmware

Experimental firmware for the Lagom keyboard

## Usage

1. Install the [`avr-hal` prerequisites](https://github.com/Rahix/avr-hal#quickstart).
2. Build firmware: `cargo build`
3. Flash to a connected board: `cargo run`
